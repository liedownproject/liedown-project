Premium streetwear landing page for LieDown Project.

## Open in VS Code

1. Open this folder in VS Code:
   `C:\Users\aley aley\Desktop\New folder (2)\liedown-vscode`

- Coming Soon section
- Email waitlist form with browser `localStorage`

- Original reference image in `assets/brand-reference.jpg`
